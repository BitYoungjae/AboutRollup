# 롤업 시작하기

## 설치하기

```shell
npm i -g rollup
```

rollup은 webpack과 다르게 별도의 cli용 모듈을 설치할 필요가 없습니다.

## 간단한 빌드

```shell
rollup -i ./src/sample1/index.js -f cjs
```

`./src/sample1/index.js` 를 Entry point 로 삼아서 `cjs` 즉 `CommonJS` 방식으로 빌드합니다.
Output 에 관련된 설정이 없으므로, `stdout` 으로 출력합니다.
